1. How to make Git “forget” about a file that was tracked but is now in gitignore?
初始化是忘了添加.gitignore, 后边添加了，如何让git不在索引已经跟踪的文件？

``` sh
git rm -r --cached .  //删除git的整个索引树
git add .    // 重建git索引
git commit -am "Remove ignored files"
```
