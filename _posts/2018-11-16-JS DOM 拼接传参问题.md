### Q
> 根据json，循环生成 table，table中 有button, click需要传入参数

1. 方法1
```js
// result is Js object
// result.name is string

result.name = "Hello";

// this is wrong, 因为生成的 html代码， Hello 是个变量名
'<input type="button" onClick="gotoNode(' + result.name + ')" />'
// 输出
<input type='button' onClick='gotoNode(Hello)'/>

// this is right
'<input type="button" onClick="gotoNode(\'' + result.name + '\')" />'
// 输出
<input type='button' onClick='gotoNode("Hello")'/>
```
